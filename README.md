# MyBlog

Локальный блог на стеке PHP 7.4 + Smarty + Bootstrap 5 + MySQL.  
**Проект адаптирован для запуска в Open Server Panel 6 (OSPanel).**

## Функционал

*   Лента постов с пагинацией.
*   Подписка на авторов и персонализированная лента.
*   Теги и фильтрация постов.
*   Система комментариев.
*   Редактирование и удаление постов (только для автора).
*   Авторизация и регистрация.

## Установка и запуск

Так как проект использует **Open Server Panel 6**, структура папок и настройки отличаются от старых версий. Следуйте инструкции строго по шагам.  
Веб‑сервер: Apache‑2.4  
PHP: PHP‑7.4  
MySQL: MySQL‑5.7  

Нажмите кнопку «Включить» (или «Старт») для этого проекта.
