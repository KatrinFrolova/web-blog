# MyBlog

Локальный блог на PHP + Smarty + Bootstrap 5 + MySQL.

## Функционал
- Лента постов
- Подписка на авторов
- Теги и фильтрация
- Комментарии
- Редактирование/удаление постов (для автора)

## Установка
1. Клонируй репозиторий.
2. Скопируй `.env.example` в `.env` и заполни параметры БД.
3. Импортируй дамп БД (schema.sql).
4. Установи права на папки cache/ и templates_c/.

## Технологии
- PHP 8.x
- Smarty 3.x
- Bootstrap 5
- MySQL
